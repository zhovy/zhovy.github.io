# Gateway 网关 JWT 鉴权

1. 用户进入网关开始登陆，网关过滤进行判断，如果是登录，则路由到后台管理微服务进行登录
2. 用户登录成功，后台管理微服务签发 JWT TOKEN 信息返回给用户
3. 用户再次进入网关开始访问，网关过滤器接收用户携带的 TOKEN
4. 网关过滤器解析 TOKEN，判断是否有权限，如果有，则放行，如果没有则返回未认证错误
